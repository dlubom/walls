WallsMap GIS Viewer and Shapefile Editor

WallsMap v0.3 (Texas Public Caves Edition)   Build date: 2014-08-20

This version of the WallsMap setup contains a preliminary GIS project named Texas Public Caves, one that's similar to the karst database being maintained by the Texas Speleological Survey. This dataset might interest you if you would like to share data with the TSS, or if you're developing a project where data of various kinds (reports, images, etc.) are associated with point locations in Texas.

Download Setup (131 MB)    Release Notes

Note: Graphics technology available only in recent versions of Microsoft Windows (Vista/7/8+) is now a system requirement. Windows XP is no longer supported.

A PDF version of the help file is also available and will be updated with each new build: WallsMap.pdf (5.5 MB)

Sample Project Info: Upon finishing, the program's setup should open a sample project file named Texas Public Caves.ntl. At the same location is an alternative project, Texas Public Caves_DRG-DEM.ntl, that has the links and settings already set up for the DEM/DRG layers described in the help file under Available Background Imagery. Those layers currently cover an area of 1177 7.5' USGS quadrangles. Before opening that project, you should download and extract the corresponding files into a folder named TSS_NTI at the same level as TSS_Data.

This program and the contained example project is offered courtesy of the Texas Speleological Survey. Once it is installed you can click on a link in the About box to check if your build is current.
